# Complete Business Analysis Project

## Project Overview
This project focuses on customer churn and sales analysis using Python.

## Business Problem
The company faced increasing churn and inconsistent revenue growth.

## Tools & Technologies
- Python
- Pandas
- Seaborn
- Scikit-learn

## Analysis Performed
- Data Cleaning
- EDA
- Hypothesis Testing
- Customer Segmentation
- Churn Prediction

## Key Insights
- High monthly charges increase churn
- Long-term customers are more loyal
- Regional sales performance varies significantly

## Recommendations
- Customer loyalty programs
- Regional marketing optimization
- Predictive churn monitoring

## Results
The analysis provided actionable recommendations to improve retention and profitability.
