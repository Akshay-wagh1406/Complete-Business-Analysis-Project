# Complete Business Analysis Project

## 📌 Project Overview
This project performs a complete business analysis using customer churn and sales datasets. The objective is to identify business problems, analyze customer behavior, evaluate sales performance, and provide data-driven recommendations.

---

## 🎯 Business Problem
The company is facing:
- High customer churn
- Declining customer retention
- Inconsistent sales performance

This project helps identify:
- Why customers leave
- Which products/regions perform best
- How to improve revenue and retention
